# boxing-game-javafx
##### ~~~~~ Boxing Game ~~~ ####

Simple Player vs CPU boxing game made with JavaFX.

*** How it works ***

* Click **Hit** to attack
* Sometimes CPU dodges
* CPU hits back if alive
* First to 0 HP = KO

*** Features ***

* Random damage + dodge
* Live HP updates
* KO win/lose message
* Reset button to restart

*** Run it ***
From project folder:

```bash
mvn javafx:run
```

*** Notes ***
Built to practice JavaFX, if/else logic, and event handling.
